UglifyJS is a battle-tested JavaScript toolkit that parses, compresses, mangles, and beautifies JavaScript code. Used by millions of projects worldwide, it reduces bundle sizes through dead code elimination, variable name mangling, and dozens of configurable compression optimizations — available as both a CLI tool and Node.js library.

## Quick example

```bash
# Install globally
npm install uglify-js -g

# Minify a file with compression and mangling
uglifyjs input.js -c -m -o output.min.js
```

```javascript
// Programmatic usage
var UglifyJS = require("uglify-js");

var result = UglifyJS.minify("function add(first, second) { return first + second; }");
console.log(result.code);
// function add(n,d){return n+d}
```

## Key features

- **50+ compress optimizations** — dead code removal, constant folding, inline expansion, and more
- **Variable and property mangling** — shorten identifiers to single characters for maximum compression
- **Source map support** — generate and compose source maps for debugging minified code
- **ES module support** — process modern ES modules with top-level `await`
- **Conditional compilation** — use `global_defs` to strip debug code at build time
- **Name cache** — maintain consistent mangled names across multiple build passes
- **SpiderMonkey AST interop** — accept input from Acorn and other parsers
